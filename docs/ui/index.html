Welcome to Pulse

Pulse is a next-generation programming language focusing on distributed systems and the Actor Model. Combining the ergonomic syntax of Python and Java with native AOT compilation, it offers both rapid scripting and raw performance.

New features include shorthand arithmetic (`+=`, `-=`), comprehensive pattern matching (`match x { 0 => println("Zero") }`), and network sockets natively supported by the standard runtime.
