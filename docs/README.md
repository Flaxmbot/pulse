# Pulse Docs (Next.js)

This is the documentation site for Pulse, built as a Next.js App Router project.

## Includes

- Beginner to advanced learning path
- Tutorials
- Grammar and language reference
- CLI, AOT/JIT, packaging, and production guides
- Compiler/runtime/self-host deep dives

## Run locally

```bash
cd docs
npm install
npm run dev
```

Open: `http://localhost:3000`

## Build

```bash
npm run build
npm run start
```
